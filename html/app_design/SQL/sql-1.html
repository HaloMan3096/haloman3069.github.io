Assignment 3 - SQL 1

This is the First section of Assignment 3

A furniture company has data pertaining to Customers, Orders, Products, and Requests

Create a relational database schema for the furniture company. Define all necessary attributes, domains, and primary and foreign keys.

  1. Each CustomerID is a number with at most three digits.
  2. Each OrderID is a number with at most five digits.
  3. Each ProductID is a number with at most two digits.
  4. Each Address is at most 50 characters long.
  5. Each Description is at most 28 characters long.
  6. Each Finish is at most 10 characters long.
  7. Quantity and Price are integer and float, respectively.

This is the Second section of Assignment 3

Include the following custom constraints in your schema:
  • Each Price is a non-negative value (zero is allowed) representing a price in dollars and cents that cannot exceed 899.99.
  • Each Quantity is a positive whole number between 1 and 50, inclusive.

This is the Third section of Assignment 3

Write insert statements to insert the data.

This is the Fourth section of Assignment 3

Display the contents of each table by adding four statements of the form: SELECT * FROM TABLE_NAME; where TABLE_NAME is substituted by the four table names above. In order to avoid conflicts, also include DROP TABLE commands for all four tables. Ensure that you drop tables in the correct order. Run the complete script and ensure the contents of the tables are displayed by your script.

SQL Script

This is the SQL script for the assignment.


DROP TABLE Customers;
DROP TABLE Orders;
DROP TABLE Products;
DROP TABLE Requests;

CREATE TABLE Customers(
    customer_id INT(3),
    name VARCHAR(255),
    address VARCHAR(50)
);

CREATE TABLE Orders(
    order_id INT(5),
    order_date VARCHAR(255),
    customer_id INT(3)
);

CREATE TABLE Products(
    product_id INT(2),
    description VARCHAR(28),
    finish VARCHAR(10),
    price Float CHECK (price < 899.99)
);

CREATE TABLE Requests(
    order_id INT(5),
    product_id INT(2),
    quantity INT CHECK (quantity >= 1 && quantity <= 50)
);
